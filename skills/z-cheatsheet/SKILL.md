# dotfiles ショートカット検索

## 引数の処理

- **引数あり** (`/z-cheatsheet 分割`): キーワードに部分一致するエントリをテーブル形式で出力する。
- **引数なし** (`/z-cheatsheet`): 全一覧をテーブル形式で出力する。

## チートシート

### WezTerm（`~/dotfiles/windows/.wezterm.lua`）

| キーワード | ショートカット | 説明 |
|-----------|--------------|------|
| リーダー, leader | `Ctrl+q` | Leader キー（1秒タイムアウト） |
| 分割, 左右分割, 横分割 | `Ctrl+q → Ctrl+q` | ペインを左右に分割 |
| 分割, 上下分割, 縦分割 | `Ctrl+q → a` | ペインを上下に分割 |
| 移動, ペイン移動, 左 | `Ctrl+←` | 左のペインに移動 |
| 移動, ペイン移動, 右 | `Ctrl+→` | 右のペインに移動 |
| 移動, ペイン移動, 上 | `Ctrl+↑` | 上のペインに移動 |
| 移動, ペイン移動, 下 | `Ctrl+↓` | 下のペインに移動 |
| 閉じる, ペイン閉じる | `Ctrl+w` | 現在のペインを閉じる |
| 全選択, 全コピー, セレクト | `Ctrl+Shift+A` | スクロールバック全体をクリップボードにコピー |

### Zsh（`~/dotfiles/wsl/.zshrc`）

| キーワード | コマンド | 説明 |
|-----------|---------|------|
| ディレクトリ, cd, ジャンプ, zoxide | `z <キーワード>` | zoxide: 履歴からスマートに cd |
| ディレクトリ, 選択, インタラクティブ, zoxide | `zi` | zoxide: fzf で履歴からディレクトリ選択 |
| 候補, 確認, 検索, query, zoxide | `zoxide query <キーワード>` | zoxide: ジャンプせずに候補リストだけ確認 |
| フォルダ, サブディレクトリ, 選択, cd | `cfd` | fzf でカレント直下のフォルダを選択して cd |

## 検索ロジック

1. ユーザーのキーワードを「キーワード」列で**部分一致検索**する。
2. 一致するエントリを以下のテーブル形式で出力する：

```
🔍 「<キーワード>」の検索結果:

| 種別 | ショートカット/コマンド | 説明 |
|------|----------------------|------|
| WezTerm | Ctrl+q → Ctrl+q | ペインを左右に分割 |
```

3. 一致するエントリがない場合、dotfiles を直接読んで検索する：
   - `~/dotfiles/windows/.wezterm.lua`
   - `~/dotfiles/wsl/.zshrc`
4. それでも見つからない場合は「該当するエントリが見つかりませんでした」と出力する。

## 注意

- **これ以上何も出力しない。コード編集・次のステップの提案は一切しない。**
