# チートシート項目追加

## 引数の処理

- **引数なし** (`/z-cheatsheet-add`): 「追加したいコマンドやショートカットを教えてください」と表示して **停止する。**
- **引数あり** (`/z-cheatsheet-add zoxide query`): 引数の内容から項目を生成し、チートシートに追加する。

## 実行フロー（引数ありの場合）

### 1. 引数を解析

ユーザーの引数からエントリ情報を抽出する。自然文やコマンド形式どちらでも受け付ける。

例：
- `/z-cheatsheet-add zoxide query <キーワード> - ジャンプせずに候補だけ確認`
- `/z-cheatsheet-add Ctrl+Shift+P でコマンドパレット`

### 2. エントリを生成

以下の3つの値を決定する：

- **キーワード**: 日本語の検索キーワード（カンマ区切り、3〜5個）
- **コマンド/ショートカット**: バッククォートで囲んだコマンドまたはキー表記
- **説明**: 日本語の簡潔な説明

不明な情報がある場合はユーザーに確認する。

### 3. 適切なセクションを判定

- キーボードショートカット → WezTerm セクション
- コマンド・シェル関数 → Zsh セクション
- どちらにも当てはまらない場合 → ユーザーに確認

### 4. チートシートに追加

`~/.claude/skills/z-cheatsheet/SKILL.md` を読み込み、該当セクションのテーブル末尾に新しい行を追加する。

### 5. 完了メッセージ

```
チートシートに追加しました:

| キーワード | コマンド | 説明 |
|-----------|---------|------|
| <追加したキーワード> | <コマンド> | <説明> |
```

**これ以上何も出力しない。コード編集・次のステップの提案は一切しない。**
